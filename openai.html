✨ 应用简介

这是一个简单易用的 OpenAI API 代理服务，基于 Deno Deploy / Cloudflare Workers 部署。 只需要一个域名和 OpenAI API Key，即可免费为家人朋友提供 AI 问答服务。

🎯 核心功能

  • 提供标准的 OpenAI API 代理端点
  • 内置精美的 Web 聊天界面
  • 支持密码保护，避免暴露 API Key
  • 流式响应，实时显示 AI 回答
  • 基于 IndexedDB 本地历史记录存储
  • 支持模型切换和自定义系统提示词
  • 集成 Tavily 搜索，为 AI 提供实时网络信息
  • 一键生成问答截图，方便分享
  • 智能会话命名，便于查找管理
